Latest Projects

SleekXMPP

During the summer of 2010, I gained co-authorship of the SleekXMPP Python library for XMPP. SleekXMPP is featured in the O'Reily book XMPP: The Definitive Guide. I have since greatly expanded the unit testing coverage, added new testing features that allow for live XMPP connections to be monitored and checked, and many other overhauls to bring the codebase to a 1.0 release-ready state.

  • Python
  • XMPP

Kestrel

What started as a group class project during the spring of 2009, has turned into an XMPP-based job dispatching and scheduling system. Tasks are distributed to worker agents based on their current presence state, allowing for rescheduling when a worker unexpectedly disconnects. Kestrel was used during the summer of 2010 by the STAR group at the Brookhaven National Laboratory for running proton collision simulations.

Related Publications

  • Kestrel: an XMPP-based Framework for Many Task Computing Applications
  • Scaling Virtual Organization Clusters over a Wide Area Network using the Kestrel Workload Management System
  • Python
  • XMPP

Weld

Weld is a basic email to XMPP gateway using Gmail's new mail notifications to simplify polling for new messages. The main use for Weld is as a link to create a SMS to XMPP gateway by converting to and from email in the middle using the SMS gateways provided by carriers. The library is written in Python and is available from PyPI.

  • Python
  • XMPP

Orion

Orion is an experiment with using HTML5's new canvas element. The goal is to create a simple multiplayer game using XMPP and JavaScript, but only the basic library for interacting with the canvas has been completed so far.

Powerware 2005-2008

Powerware was a content management and resource tracking system for that I developed for church ministries. At the start in 2005, it used a custom MVC PHP framework but was eventually migrated to CodeIgniter. The system provided an event calendar, work schedules, room and vehicle reservations, and team membership management. The information stored in the MySQL database used by Powerware was used to also run a public facing web site.

Over its three year lifespan, Powerware was used by four different organizations until Facebook made it obsolete.
